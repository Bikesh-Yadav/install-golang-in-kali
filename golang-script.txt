


	INSTALL GO LANG IN LINUX
--------------STEPS------------

1. CHECK IF GOLANG IS INSTALLED
	> go version

2. IF NOT INSTALLED, 
	ðŸ”¹ Option 1: From Kali's Package Manager
		
		>sudo apt update
		>sudo apt install golang


	ðŸ”¹ Option 2: Install Latest from Official Source
		1.Download latest Go from https://golang.org/dl/
		2.Example Latest Version for: Go is 1.25.5:
		
			>wget https://go.dev/dl/go1.24.5.linux-amd64.tar.gz
			>sudo rm -rf /usr/local/go
			>sudo tar -C /usr/local -xzf go1.24.5.linux-amd64.tar.gz
		
		
		3.Add to PATH (if not already):
			>echo 'export PATH=$PATH:/usr/local/go/bin' >> ~/.bashrc
			>source ~/.bashrc
			
		4. Then verify again:
			> go version
